## How-To Use
**Clone this repo**

Run three.js project

1. Put three folder onto a webserver
2. Navigate to page

Run/Build Unity project

1. Install Unity
2. Open Unity project
3. Press play
4. Build WebGL

Build Cardboard project

1. Build Android
